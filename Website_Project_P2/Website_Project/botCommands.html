GooseBot Commands:

Commands: Description Parameters
.help Gooseneck bot gives a brief overview of the different commands that you can use and how to start using it none
.help cmd in depth information about commands and the bot itself none
.home Gooseneck bot sends a hyperlink that sends the user to The GooseBot home page! none
.start This command starts the Gacha game. Once the user types this, they will be able to roll for cards and Gooseneck Bot begins to store the users data. none
.roll Runs two random functions, one for which card the user is going to get and another for the cards rarity. none
.info This command will display information about a specific card you would like to see/know more about. Card Name
